# QuantLab Reviews Repository

This repository stores external review artifacts for QuantLab.

Main project repository:
mikeaccio666-debug/xiami-quantlab

This repository is for review artifacts only.

Allowed:
- Xiami review packets
- Hermes risk reviews
- External agent handoff packets

Forbidden:
- Trading code
- Backtest engine code
- Broker integrations
- Execution modules
- API keys
- Credentials
- Private keys
- Tokens
- Raw market data
- Licensed data

Review artifacts do not approve:
- Milestone 1
- Paper trading
- Live trading
- Broker integration
- Capital allocation
- Strategy promotion

Only the human owner can approve phase or milestone changes in the main QuantLab repository.
